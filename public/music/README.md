# Nhạc nền cho thiệp

Đặt các file MP3 vào thư mục này để dùng trong thiệp:

- **tet.mp3** – Nhạc nền Tết
- **valentine.mp3** – Nhạc nền Valentine  
- **womensday.mp3** – Nhạc nền 8/3

Sau khi thêm file, trong admin chọn "Nhạc nền Tết" / "Nhạc nền Valentine" / "Nhạc nền 8/3" khi tạo hoặc sửa thiệp.  
Khi xem thiệp, bấm nút **Phát nhạc** trên trang để nghe (trình duyệt yêu cầu bấm trước khi phát).
